An HTML5/JavaScript recreation of the Jurassic Park control system.

View the source on GitHub

About

Jurassic Systems is an HTML5 and JavaScript based recreation of the IRIX and Macintosh environments featured in the original Jurassic Park film. The recreation focuses on the scene where Ray Arnold attempts to circumvent Dennis Nedry's formidable security system while an Elvis-bodied Nedry looks on.

Created by Tully Robinson.

Features

  • Working command line with the following commands:
    Command Summary
    accessAccess a target environment on the Jurassic Systems grid.
    systemCheck a system's current status.
    lsList files in the current directory.
    displayDisplay image files (hint: use ls to find a "file").
    musicTurn background music on or off.
    keychecksDisplay system level command history.
    manDisplay reference manual for a given command.
    helpList all available commands.
  • Three incorrect access attempts trigger the Nedry system lock down.
  • Simulated desktop environment with system sounds from the movie.
  • One hidden feature.
